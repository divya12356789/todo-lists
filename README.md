# To-Do List Web App 📝

A simple To-Do List web application built using HTML, CSS, and JavaScript.

## 📌 Features
- Add tasks
- Remove tasks
- Clean and minimal design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## 🖥 How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Start adding and removing your tasks!
